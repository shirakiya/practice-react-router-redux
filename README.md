# practice-react-router-redux
My react-router-redux practice.


# How to run
### install libraries
```
$ pip install -r requirements.txt
$ npm install
```

### run app
```
$ flask run
$ npm start
```


# Environment Variables
| name        | value    |
|-------------|----------|
| FLUSK_APP   | "app.py" |
| FLUSK_DEBUG | 0 or 1   |
